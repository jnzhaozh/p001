# P001
This repository contains R code to reproduce simulations and figures for:
"Collective Behavior with Heterogeneous Information Exposure" (Zhao).

## Files
- v2-1.R to v2-5.R: scripts for Sim-1 to Sim-5 (and/or corresponding figures)
- theme_this.R: plotting theme helper

## How to run
1. Install required R packages (see below).
2. Run scripts in order: v2-1.R, v2-2.R, v2-3.R, v2-4.R, v2-5.R.

## Requirements
R (>= 4.1 recommended). Packages used include:
tidyverse, ggplot2, truncnorm (and any others loaded in the scripts).
